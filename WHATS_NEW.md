# 🎉 flomo 笔记 - 功能完善版发布！

## ✨ 新增功能

### 1. 分享功能 📤
**可以分享笔记啦！**
- 📋 复制内容到剪贴板
- 📱 分享到微信/QQ/钉钉
- 💾 导出为文本文件

**如何使用：**
1. 首页找到想分享的笔记
2. 点击笔记右下角的 📤 图标
3. 选择复制、分享或导出

---

### 2. 深色模式 🌙
**护眼模式来了！**
- 🌓 深色/浅色主题切换
- 📱 跟随系统设置
- 💾 自动保存偏好

**如何使用：**
1. 点击右上角 ⚙️ 设置图标
2. 进入设置页面
3. 切换"深色模式"开关

---

### 3. 设置页面 ⚙️
**应用设置中心！**
- 🎨 外观设置（深色模式）
- 📤 数据管理（导出/导入）
- ℹ️ 关于信息
- ❓ 使用帮助

**如何访问：**
- 首页右上角 ⚙️ 图标

---

### 4. 数据导出 💾
**备份你的笔记！**
- 📦 导出所有笔记为 JSON
- 📊 包含完整元数据
- 🔒 本地保存，安全私密

**如何使用：**
1. 设置 → 数据管理
2. 点击"导出笔记"
3. 选择保存位置

---

### 5. 每日回顾 📖
**温故而知新！**
- 📝 查看今天的笔记
- 🎲 随机回顾历史笔记
- 💪 激励语录
- 📊 统计摘要

**如何使用：**
- 开发中（即将上线）

```bash
# 1. 登录 Expo
eas login

# 2. 配置项目
eas build:configure

# 3. 开始构建
eas build --platform android --profile preview
```
