## 📝 使用指南

### 快速开始
1. 打开 FlowMo
2. 点击右下角蓝色 + 按钮
3. 输入笔记内容
4. 使用工具栏添加格式
5. 点击"预览"查看效果
6. 添加标签（可选）
7. 点击"保存"

### Markdown 语法
```
**粗体** → **text**
*斜体* → *text*
列表 → - item
待办 → - [ ]
引用 → > text
代码 → `code`
链接 → [text](url)
```
