  - *斜体* - 实时渲染为斜体

  - - [ ] 待办 - 实时渲染为复选框
  - > 引用 - 实时渲染为引用块
  - `代码` - 实时渲染为行内代码
  - ``` 代码块 - 实时渲染为代码块
  - [链接](url) - 实时渲染为可点击链接
  - ![图片](url) - 支持图片

## 🎨 Markdown 语法说明

### 标题
```
# 一级标题
## 二级标题
### 三级标题
```

### 强调
```
**粗体**
*斜体*
~~删除线~~
```

### 列表
```
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2

- [ ] 待办事项
- [x] 已完成
```

### 引用和代码
```
> 这是一段引用

行内代码 `console.log()`

```javascript
// 代码块
function hello() {
  console.log('Hello World');
}
```
```

### 链接和图片
```
[链接文字](https://example.com)

![图片描述](https://example.com/image.jpg)
```

## 💡 使用说明

### 快速开始
1. 打开 FlowMo
2. 点击右下角蓝色 + 按钮
3. 输入笔记内容
4. 使用工具栏或输入 Markdown 语法
5. 点击"预览"查看渲染效果
6. 添加标签（可选）
7. 点击"保存"

### Markdown 快捷输入
- **工具栏**: 点击工具栏按钮快速插入语法
- **自动补全**: 输入 `- ` 自动创建列表
- **实时预览**: 点击右上角眼睛图标查看效果
