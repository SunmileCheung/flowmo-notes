# 📖 flomo 笔记 - 使用指南

## 🎯 快速上手（3 分钟）

### 第一步：启动应用

```bash
cd flomo-clone
npm install
npx expo start
```

手机下载 **Expo Go**，扫描二维码。

## 💡 使用技巧

### ✨ 高效记录

1. **快速捕捉灵感**
   - 想到什么记什么
   - 不要纠结格式
   - 标签后续可加

2. **标签使用建议**
   - 保持标签简洁（2-4 字）
   - 不要过度分类
   - 自然形成标签体系

3. **定期回顾**
   - 每周查看统计
   - 每月整理标签
   - 每季度回顾内容

### 🎯 最佳实践

**✅ 推荐做法：**
- 每天记录 3-5 条
- 使用 2-3 个标签
- 每周回顾一次
- 保持简洁

**❌ 避免做法：**
- 过度分类
- 追求完美格式
- 标签过多过细
- 长时间不回顾

## 🔧 常见问题

### Q: 数据会丢失吗？
A: 数据存储在本地，卸载应用会丢失。建议定期导出备份。

### Q: 支持多设备同步吗？
A: 当前版本仅支持单设备本地存储。

### Q: 可以导入 flomo 数据吗？
A: 暂不支持，但可以自己编写脚本导入。

### Q: 如何备份数据？
A: 数据在 AsyncStorage 中，可通过代码导出为 JSON。

### Q: 支持图片吗？
A: 当前版本仅支持文本，图片功能待开发。

## 🎨 自定义指南

### 修改主题色

编辑 `src/screens/HomeScreen.js`：
```javascript
// 将 #007AFF 改为你喜欢的颜色
backgroundColor: '#007AFF'  // 蓝色
backgroundColor: '#34C759'  // 绿色
backgroundColor: '#FF9500'  // 橙色
```

### 修改应用名称

编辑 `app.json`：
```json
{
  "expo": {
    "name": "你的应用名称",
    "slug": "your-app-name"
  }
}
```

### 添加新功能

参考 `src/screens/` 目录下的现有代码结构，复制并修改。
