## 💡 使用说明

### 快速开始
1. 打开 FlowMo
2. 点击右下角蓝色 + 按钮
3. 输入笔记内容（支持 Markdown）
4. 实时看到渲染效果
5. 添加标签（可选）
6. 点击"保存"

### Markdown 语法
```
# 一级标题
## 二级标题

**粗体**
*斜体*

- 无序列表
1. 有序列表

- [ ] 待办事项
- [x] 已完成

> 引用内容

`行内代码`
```
